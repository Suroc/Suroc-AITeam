# Phase 1 MVP - 产品需求文档（PRD）

更新时间：2026-03-26 16:12 CST
项目：AI 工具导航与需求收集站（MVP）
状态：draft

## 1. 项目目标
构建一个轻量级站点，用于：
1. 展示 AI 工具目录
2. 按分类检索工具
3. 提供工具详情查看
4. 收集用户需求与反馈
5. 支撑后续 AI 团队对工具产品方向的判断

## 2. 目标用户
1. 对 AI 工具感兴趣的个人用户
2. 需要寻找工具方案的企业用户
3. 需要提交需求的潜在合作方

## 3. 核心功能
### 3.1 工具列表页
- 展示工具卡片
- 支持基础分类
- 支持简单搜索

### 3.2 工具详情页
- 展示工具简介
- 展示能力标签
- 展示适用场景
- 提供需求反馈入口

### 3.3 需求提交页
- 提交姓名/联系方式
- 提交需求描述
- 选择需求类型

### 3.4 简易后台
- 工具条目录入
- 需求记录查看

## 4. MVP 边界
### 纳入范围
1. 基础前台展示
2. 基础需求提交
3. 基础后台录入/查看

### 暂不纳入
1. 用户登录体系
2. 支付系统
3. 推荐算法
4. 复杂数据分析

## 5. 验收标准
1. 前台可正常浏览工具列表与详情
2. 用户可成功提交需求
3. 后台可录入工具、查看需求
4. 基础页面与接口可联通
5. 可部署为可访问版本
