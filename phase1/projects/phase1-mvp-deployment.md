# Phase 1 MVP - 部署建议

更新时间：2026-03-26 16:14 CST
项目：AI 工具导航与需求收集站（MVP）
状态：active

## 部署目标
以最小复杂度完成首个 MVP 的可访问部署。

## 建议方案
1. 前后端一体化优先
2. 优先使用轻量部署平台
3. MVP 期优先保证可上线与可验证，不追求复杂基础设施

## 部署建议路径
1. 使用 Next.js 一体化项目结构
2. 使用 SQLite 或轻量数据库
3. 使用 Vercel / 轻量 Node 环境进行部署

## 发布前要求
1. 页面与 API 已联通
2. 基础测试通过
3. 关键表单提交通路正常
4. 有简单错误排查说明
