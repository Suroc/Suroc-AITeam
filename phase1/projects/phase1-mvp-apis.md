# Phase 1 MVP - API 清单

更新时间：2026-03-26 16:14 CST
项目：AI 工具导航与需求收集站（MVP）
状态：active

## Tool APIs
1. GET /api/tools
   - 获取工具列表

2. GET /api/tools/:id
   - 获取工具详情

3. POST /api/admin/tools
   - 新增工具

4. PUT /api/admin/tools/:id
   - 编辑工具

## Requirement APIs
5. POST /api/requirements
   - 提交需求

6. GET /api/admin/requirements
   - 获取需求列表

7. GET /api/admin/requirements/:id
   - 获取需求详情
